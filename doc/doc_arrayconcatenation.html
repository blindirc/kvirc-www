Array concatenation operator
This operator concatenates scalars, arrays or hashes to a target array - if the target is not an array, it is converted to one first.
The syntax is:

<target> <+ <right_operand>

If <right_operand> is a scalar then it is appended to <target>, otherwise the constituent items of the array or values of the hash are appended.
